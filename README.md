## Motivation

I'm thrilled to embark on the journey of learning three.js and delving into the captivating world of 3D graphics using React.

## Tech Stack:

- React.
- Three JS.
- React Three Fiber.
- Tailwind CSS.
- Vite

## Getting Started

1. Clone the repository
2. `npm install`
3. `npm run dev`
4. That's it!
